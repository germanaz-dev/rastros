> *El alquiler vence el viernes. Tienes $4.200 en la cuenta y un fax que lleva dos días escupiendo papel.*

Juego de detective privado generado por inteligencia artificial. Cada caso es único — ciudades reales, ladrones inventados, pistas culturalmente específicas. Sin servidor. Sin registro. Un solo archivo HTML.

---

## Cómo jugar

1. Descarga `index.html` de la carpeta `v3-current/`
2. Abre el archivo en cualquier navegador moderno
3. Consigue una API key gratuita de Gemini en [aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)
4. Ponle nombre a tu investigador y empieza

La API key se guarda solo en tu dispositivo. No pasa por ningún servidor.

---

## Mecánicas

- 🌍 **20 ciudades** en todo el mundo con flavor text propio
- 🤖 **Casos generados por IA** — nunca se repiten
- 💰 **Presupuesto real** — los vuelos, hoteles y análisis forenses salen de tu bolsillo
- ⭐ **Sistema de puntos** independiente del dinero — reputación acumulada
- ⚡ **Confrontación directa** — sin orden de arresto, +50% de bonus si aciertas, penalización si fallas
- 🗺️ **Mapa animado** estilo Indiana Jones al viajar entre ciudades
- 🔬 **Tech tree** — de intuición básica a análisis de ADN portátil
- 📋 **M.A.L.O.S.** — organización criminal ficticia con casos conectados
- 💾 **Código de partida** exportable para continuar en otro dispositivo
- 📱 **Responsive** — funciona en móvil, tablet y escritorio

## Requisitos técnicos

- Navegador moderno (Chrome, Firefox, Safari, Edge)
- API key de Google Gemini (gratuita)
- Sin instalación, sin dependencias, sin servidor

El juego usa **Gemini 3.1 Flash Lite Preview** como modelo principal con fallback automático a modelos anteriores si se agota la cuota diaria.

---

## API Key de Gemini

1. Ve a [aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)
2. Inicia sesión con tu cuenta de Google
3. Crea una nueva API key
4. Pégala en la pantalla de inicio del juego

El tier gratuito incluye 500 requests/día — más que suficiente para jugar.
